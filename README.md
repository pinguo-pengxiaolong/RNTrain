# RNTrain
RN简单TODO项目，此项目也可以作为RN启动项目

#### 特征
  - [x]集成Native Navigation(Pod)
  - [x]集成现有项目(Pod)
  - [x]使用最新版的RN与React Native
  - [ ]TypeScript语法支持(todo)

#### 安装

```sh
	>> npm install
	>> pod install
```
- 注意安装过程会出现Animation动画库引用错误，找到错误的那一行，修改为：`import <React/RCTValueAnimatedNode.h>`

#### 使用

``` sh
	// 找到RNTrain.xcworkspace，Xcode启动
	// 然后命令行执行:
	>> sh build.sh
```
